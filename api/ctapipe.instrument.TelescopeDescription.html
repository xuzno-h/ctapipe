TelescopeDescription¶

class ctapipe.instrument.TelescopeDescription(name, type, optics: ctapipe.instrument.optics.OpticsDescription, camera: ctapipe.instrument.camera.CameraGeometry)[source]¶

Bases: object

Describes a Cherenkov Telescope and it’s associated OpticsDescription and CameraGeometry

The string representation is a combination of the optics and camera, separated by a colon: “optics:camera” (e.g. “SST-1m:DigiCam”)

The TelescopeDescription.guess() constructor can be used to fill in info from metadata, e.g. for Monte-Carlo files.

Parameters
name: str

Telescope name

type: str

optics: OpticsDescription

the optics associated with this telescope

the camera associated with this telescope
