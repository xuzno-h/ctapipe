TableWriter¶

class ctapipe.io.tableio.TableWriter(parent=None, add_prefix=False, **kwargs)[source]¶

Bases: ctapipe.core.component.Component

Base class for writing Container classes as rows of an output table, where each Field becomes a column. Subclasses of this implement specific output types.

write(table_name, containers)[source]¶

Write the contents of the given container or containers to a table. The first call to write will create a schema and initialize the table within the file. The shape of data within the container must not change between calls, since variable-length arrays are not supported.

Parameters
table_name: str

name of table to write to

container: `ctapipe.core.Container`

container to write
